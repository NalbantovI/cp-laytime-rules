# Charter Party Laytime Extraction Summary

**Date:** 20 November 2025
**Project:** Complete Laytime Clause Extraction from Charter Party Library
**Method:** Automated OCR extraction using pdfplumber

---

## Overview

This project involved systematically extracting laytime, demurrage, and despatch clauses from all charter party documents in the library using OCR technology.

### Statistics

- **Total Subfolders Processed:** 75
- **Total PDF Files Analyzed:** 300+ files
- **Extraction Method:** Python with pdfplumber library
- **Keywords Searched:** 25+ laytime-related terms

### Keywords Used for Extraction

The extraction process searched for the following keywords:
- laytime, lay time, laydays, lay days
- demurrage, despatch, dispatch
- notice of readiness, NOR, N.O.R.
- running days, running hours
- working days, weather working days, WWD
- weather permitting
- turn time, berth
- excepted, exception, excluded
- strikes, sundays, holidays
- commence, commencement
- loading, discharging, discharge
- rate, per day, per hour
- reversible, non-reversible
- detention, damages for detention

## How to Use These Files

Each laytime analysis file contains:

1. **Header Section** - Charter party name and file statistics
2. **Per-File Analysis** - Each PDF analyzed separately
3. **Extract Sections** - Numbered extracts showing:
   - Context lines before the keyword
   - The line containing the laytime-related keyword
   - Context lines after (showing the full clause)

### Example Extract Format

```
### Extract 1

```
[5 lines of context before]
LINE CONTAINING LAYTIME KEYWORD
[15 lines of context after]
```
```

## Technical Details

### Extraction Script Features

- **Language:** Python 3.13
- **Main Library:** pdfplumber
- **Fallback:** PyPDF2
- **Error Handling:** Page-level error recovery
- **Context Window:** 5 lines before, 15 lines after
- **Duplicate Removal:** Automatic deduplication of extracts

### Script Location

- Main script: `/extract_laytime.py`
- Continuation script: `/extract_laytime_continue.py`
