## Common Rule Patterns

### Pattern 1: "Time Not to Count" Structure
**Frequency:** ~100+ rules

Standard format:
- Condition described (e.g., "shifting from anchorage")
- "shall not count as laytime"
- Often adds "even if vessel is on demurrage"
- Sometimes adds cost allocation

### Pattern 2: "Unless Used" Exception
**Frequency:** ~30+ rules

Standard format:
- Period normally excluded (e.g., "Sundays and holidays excluded")
- "unless used"
- "in which case actual time used to count"

### Pattern 3: "WIBON/WIPON" Commencement
**Frequency:** ~20+ rules

Standard format:
- Laytime to commence after NOR
- "whether in berth or not"
- "whether in port or not"
- "whether in free pratique or not"
- "whether customs cleared or not"

### Pattern 4: "Weather Working Day"
**Frequency:** ~25+ rules

Standard format:
- Rate specified (e.g., "10,000 MT per WWWD")
- "weather permitting"
- Sometimes "Sundays and holidays included/excluded"

### Pattern 5: "Pro-Rata" Calculations
**Frequency:** ~15+ rules

Standard format:
- Time counting provision
- "pro-rata according to [basis]"
- Often related to equipment availability

## Key Takeaways

3. Vessel unreadiness/equipment failure

### Most Common Provisions:
1. NOR tendering requirements
2. WIBON/WIPON clauses
3. "Unless used" exceptions
4. "Even if on demurrage" qualifications
5. Cost allocation (Owner's vs Charterer's account)

- **Iron Ore Charters** (VALE, CSP, SAMARCO): Similar provisions, focus on draft surveys
- **Coal Charters** (RTS, RTM, YANCOAL): Emphasis on hold cleanliness, equipment
- **Grain Charters** (NORGRAIN, SYNACOMEX): Weather provisions, trimming time
